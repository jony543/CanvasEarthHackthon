Module.InkCanvas

Creates ink canvas connected with html canvas. This layer is the target of screen rendering.

new InkCanvas(canvas, width, height, webGLContextAttributes)

Parameters:
Name Type Description
canvas HTMLCanvasElement DOM element related with ink canvas
width int initial width
height int initial height
webGLContextAttributes Module.WebGLContextAttributes WebGL context configuration

Properties

Name Type Description
surface HTMLCanvasElement Drawing surface
ctx WebGLRenderingContext Drawing surface context
width int Layer width
height int Layer height
bounds Module.Rectangle The bounds of the layer. Equals to (0, 0, width, height).

Methods

requestAnimationFrame(callback, present)

Adds callback to queue, executed on first animation frame. This method is not applicable when preserveDrawingBuffer WebGL context attribute is setted.

Parameters:
Name Type Description
callback function A parameter specifying a function to call when it's time to update your animation for the next repaint.
The callback has one single argument, a DOMHighResTimeStamp,
which indicates the current time (the time returned from Performance.now()) for when requestAnimationFrame starts to fire callbacks.
present boolean is callback affects canvas
Returns:
  • int animation frame ID

drawStroke(brush, path, width, color, roundCapBeginning, roundCapEnding, ts, tf, drawContext)

Draws points with a brush over the layer

Parameters:
Name Type Description
brush Module.StrokeBrush instance of the Module.StrokeBrush class that determines how the stroke is going to be rendered
path Module.Path points to draw
width float The width of the stroke. If the stroke has a variable thickness which is included in the control points array, this parameter must be NaN.
color Module.Color stroke color
If the stroke has a variable alpha which is included in the control points array, alpha component must be NaN.
roundCapBeginning boolean Cap the stroke with a circle at the start
roundCapEnding boolean Cap the stroke with a circle at the end
ts float The starting value for the Catmull-Roll spline parameter
tf float The final value for the Catmull-Rom spline parameter
drawContext Module.StrokeDrawContext Module.StrokeDrawContext instance (contains previous path point), applicable only when brush is Module.ParticleBrush
Returns:
  • Module.Rectangle affected area
