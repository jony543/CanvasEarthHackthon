enum BlendMode

Enum for blend modes

Type:
  • Object
Properties:
Name Type Description
NORMAL Object Standard alpha composition
NORMAL_REVERSE Object Like NORMAL, but the drawn layer goes at the back
ERASE Object The opacity (alpha) of the draw layer specify how the pixel of the target to be cleared
NONE Object No color mixing, the result is the layer drawn
MAX Object The result is the maximum of both color. The result is a lighter color.
MIN Object The result is the minimum of both color. The result is a darker color.
ADD Object Sums the color components
SUBSTRACT Object Subtract the destination color components from the drawn color components
SUBSTRACT_REVERSE Object Subtract the drawn color components from the destination color components
MULTIPLY_NO_ALPHA Object Plain multiplication, alpha is multiplied as well
MULTIPLY_NO_ALPHA_INVERT Object Equals to 1 - MULTIPLY_NO_ALPHA

enum PropertyFunction

Enumeration of the properties of a path, constructed by the Module.PathBuilder subclasses

Type:
  • Object
Properties:
Name Type Description
Power Object The function f(x) = x^a. When a is 1.0, the function becomes f(x) = x, the identity.
Periodic Object Periodic function that cycles between the minimum and maximum value. The parameter sets the number of half-cycles.
When the parameter is odd number, the function will begin with the minimum value and will finish with the maximum value.
When period is even number the function starts and finish with the same value.
Sigmoid Object A sigmoid function is a function having an "S" shape. It goes from the minimum to the maximum value.
It accelerates from the start to the middle and then decelerates to the end. The parameter controls the acceleration.

WebGLContextAttributes

Defines WebGL context attributes

Type:
  • Object
Properties:
Name Type Description
alpha, defaults: true boolean If true, request an alpha channel for the context. If you create an alpha channel, you can blend the canvas rendering with the underlying web page contents.
depth, defaults: true boolean If true, request a depth buffer of at least 16 bits. If false, no depth buffer will be initialized.
stencil, defaults: false boolean If true, request a stencil buffer of at least 8 bits. If false, no stencil buffer will be initialized.
antialias, defaults: true boolean If true, antialiasing will be initialized with a browser-specified algorithm and quality level. If false, antialiasing is disabled.
premultipliedAlpha, defaults: true boolean If true, the alpha channel of the rendering context will be treated as representing premultiplied alpha values.
If false, the alpha channel represents non-premultiplied alpha.
preserveDrawingBuffer, defaults: false boolean If true, the contents of the drawing buffer are preserved between consecutive requestAnimationFrame() calls.
If false, color, depth and stencil are cleared at the beginning of each requestAnimationFrame(). Generally setting this to false gives better performance.
preferLowPowerToHighPerformance, defaults: false boolean If true, hints the browser to initialize a low-power GPU rendering context.
If false, prefers to initialize a high-performance rendering context.
failIfMajorPerformanceCaveat, defaults: false boolean If true, requests context creation to abort if the browser is only able to create a context
that does not give good hardware-accelerated performance.

OffscreenLayerOptions

OffscreenLayer definition

Type:
  • Object
Properties:
Name Type Description
width int layer width, required when user defined framebuffer or renderbuffer properties are not available or specific texture size is desired.
height int layer height, required when user defined Framebuffer or Renderbuffer properties are not available or specific texture size is desired.
scaleFactor float layer is scaled
texture WebGLTexture user defined texture, width and height not required when desired size matches image size and image is attached to the texture (Module.GLTools.prepareTexture attaches image).

renderbuffer WebGLRenderbuffer user defined renderbuffer. Renderbuffer dimensions used for layer creation.
ownGlResources boolean if true layer will free the GL resources when deallocated, applicable when there is user defined GL resource
